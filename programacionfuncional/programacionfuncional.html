Lambda

Importante

No se puede asignar una lambda a una referencia var

//Error:

//cannot infer type for local variable lambda:

var lambda=s->s+2; //No compila

var consumer = System.out::print; //No compila

Importante

No se puede mezclar en los parametros de las lambdas var y tipos

//Error:

Comparator c = (var s1, String s2) -> 0; interface I3 extends I2 {void m3();}

Reglas para declarar Lambdas validas

  • Si hay mas de un parametro en la parte izquierda o no hay, deben estar encerrados en parantesis

    • a -> a*a; //Correcto se puede obviar los parentesis con un solo parametro

    int a -> a*a; //Correcto

    () -> System.out.println("Hola"); //Correcto

    (int a, int b) -> a+b; //Correcto

    int a, int b -> a+b; //No compila

    a,b -> a+b; //No compila

Interfaz funcional

Importante

Una interfaz funcional es aquella que solamente posee un metodo abstracto, el objetivo de las expresiones lambda es una interfaz funcional.

interface I1 { void m1();} //Interfaz funcional

interface I2 extends I1 {void m2();} //Interfaz no es funcional

interface I3 extends I2 {void m3();} //Interfaz no es funcional

interface I4 { String toString(); } //Interfaz no es funcional
